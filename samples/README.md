# samples/

Pliki do recznych testow ekstrakcji. Trzymaj tu m.in.:

- plik natywny (np. `probka.docx` lub tekstowy PDF) — sprawdza sciezke PDFBox/POI,
- skan po polsku (np. `skan-pol.png` lub PDF bedacy skanem) — sprawdza OCR + pakiet `pol`.

Zawartosc (poza tym README i `.gitkeep`) jest ignorowana przez git — moga tu byc
dane wrazliwe, ktore nie powinny trafic do repozytorium.
